üöÄ Getting Started  
### Prerequisites  
- Python 3.8+  
- Docker (for deployment)  
- Redis (for real-time processing)  

### Installation  
```bash
# Clone repository
git clone https://github.com/Mr-Usmanprof/fraud-detection-engine.git

# Launch Jupyter Lab
jupyter lab
```
